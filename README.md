# Finance Tracker

## Tech Stack

- React
- TypeScript
- Supabase

## To run the project

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## To connect to Supabase

- https://supabase.com/
- add you credentials to the .env file

## To migrate the database

- https://supabase.com/docs/guides/deployment/database-migrations
